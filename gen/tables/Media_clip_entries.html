Table 273 – Entries common to all media clip dictionaries

Key

Type

Value

Type

name

(Optional) The type of PDF object that this dictionary describes; if present, shall be MediaClip for a media clip dictionary.

(Required) The subtype of media clip that this dictionary describes. May be MCD for media clip data (see 13.2.4.2, “Media Clip Data” ) or MCS for a media clip section (see 13.2.4.3, “Media Clip Section” ). The media clip shall be considered non-viable if the conforming reader does not recognize the value of this entry.
