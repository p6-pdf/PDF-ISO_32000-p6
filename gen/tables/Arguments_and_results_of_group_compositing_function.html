Table 141 – Arguments and results of the group compositing function

Variable

Meaning

G

The transparency group: a compound object consisting of all elements E 1 , … , E n of the group—the n constituent objects’ colours, shapes, opacities, and blend modes

C0

Colour of the group’s backdrop

C

Computed colour of the group, which shall be used as the source colour when the group is treated as an object

f

Computed shape of the group, which shall be used as the object shape when the group is treated as an object

0

Alpha of the group’s backdrop

Computed alpha of the group, which shall be used as the object alpha when the group is treated as an object
