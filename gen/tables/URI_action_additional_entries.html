Table 206 – Additional entries specific to a URI action

Key

Type

Value

S

name

(Required) The type of action that this dictionary describes; shall be URI for a URI action.

URI

ASCII string

(Required) The uniform resource identifier to resolve, encoded in 7-bit ASCII.

IsMap

boolean

(Optional) A flag specifying whether to track the mouse position when the URI is resolved (see the discussion following this Table). Default value: false.

This entry applies only to actions triggered by the user’s clicking an annotation; it shall be ignored for actions associated with outline items or with a document’s OpenAction entry.
